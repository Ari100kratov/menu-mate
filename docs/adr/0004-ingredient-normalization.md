# 0004 Осторожная нормализация ингредиентов

## Статус

Принято.

## Контекст

Список покупок должен суммировать совместимые ингредиенты и не терять неоднозначную информацию из рецепта.

## Решение

На первом этапе нормализуем только безопасные единицы:

- кг в г;
- л в мл;
- точные штуки с точными штуками.

Бытовые меры вроде ложек, упаковок и щепоток остаются отдельными, пока нет продуктовой таблицы конверсий.

## Последствия

- Список покупок объединяет меньше позиций, но делает это безопаснее.
- Неопределенные количества сохраняются отдельно.
- Будущие словари конверсий можно добавить без изменения данных рецепта.
