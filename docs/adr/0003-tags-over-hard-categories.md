# 0003 Теги вместо жестких категорий рецептов

## Статус

Принято.

## Контекст

Рецептам нужны гибкая фильтрация, поиск, рекомендации и будущая автоматическая классификация.

## Решение

Используем теги вместо жестких категорий рецептов. Теги имеют нормализованные названия и могут быть системными, пользовательскими или предложенными.

## Последствия

- Карточка рецепта может показывать несколько тегов.
- Дедупликация идет по нормализованному имени.
- Категории продуктов остаются для группировки списка покупок, но не являются категориями рецептов.
