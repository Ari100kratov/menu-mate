# 0003 Теги вместо жестких категорий рецептов

## Статус

Частично заменено [ADR-0014](0014-shared-product-catalog-and-recipe-category.md).

## Контекст

Рецептам нужны гибкая фильтрация, поиск, рекомендации и будущая автоматическая классификация.

## Решение

Используем теги для гибкой дополнительной классификации. Теги имеют нормализованные названия и могут быть системными, пользовательскими или предложенными. ADR-0014 позднее добавил одну основную категорию блюда; теги не заменяют её.

## Последствия

- Карточка рецепта может показывать несколько тегов.
- Дедупликация идет по нормализованному имени.
- Категории продуктов остаются для группировки списка покупок и не совпадают с основной категорией блюда.
