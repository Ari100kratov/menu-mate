# 0007 Локальный запуск через Aspire

## Статус

Принято.

## Контекст

MenuMate нужен надежный локальный entry point, который поднимает API и всю обязательную инфраструктуру вместе.

## Решение

Используем `MenuMate.AppHost` как основной локальный запуск. AppHost поднимает PostgreSQL, MinIO, `minio-init`, мигратор, API и Vite-фронтенд.

## Последствия

- Разработчик запускает один проект.
- API стартует после применения миграций.
- Изображения доступны через локальный MinIO.
- Frontend получает фактический адрес API через Aspire и Vite proxy.
