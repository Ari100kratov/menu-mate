# 0006 Черновики импорта

## Статус

Принято.

## Контекст

Импорт из изображений и других недоверенных источников может быть неточным. Прямое сохранение извлечённых данных может испортить библиотеку рецептов.

## Решение

RecipeImports создает черновики. Черновик может содержать исходный текст, предложенные ингредиенты, шаги, теги и доказательства извлечения. Только явное подтверждение пользователя создает или обновляет рецепт.

## Последствия

- RecipeImports зависит от Recipes через commands/contracts, а не через прямую мутацию сущностей.
- Raw extraction evidence помогает пользователю исправлять ошибки.
- Правило "не сохранять без подтверждения" остается enforceable.
- Черновики и исходные изображения автоматически удаляются через настраиваемый срок после последнего изменения; по умолчанию срок равен 7 дням.
