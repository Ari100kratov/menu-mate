# 0006 Черновики импорта

## Статус

Принято.

## Контекст

Импорт из screenshot или URL может быть неточным. Прямое сохранение извлеченных данных может испортить библиотеку рецептов.

## Решение

Imports создает черновики. Черновик может содержать исходный текст, предложенные ингредиенты, шаги, теги и доказательства извлечения. Только явное подтверждение пользователя создает или обновляет рецепт.

## Последствия

- Imports зависит от Recipes через commands/contracts, а не через прямую мутацию сущностей.
- Raw extraction evidence помогает пользователю исправлять ошибки.
- Правило "не сохранять без подтверждения" остается enforceable.
